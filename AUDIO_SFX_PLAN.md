# Speed Dungeon - Sound Effects Plan

A categorized list of every game action and UI interaction that would benefit from audio feedback.

---

## Combat - Attack Sounds

| Action | Notes |
|--------|-------|
| **Sword swing** (1H/2H) | Whoosh — different weight for one-hand vs two-hand |
| **Stab** (1H/2H) | Sharper, quicker thrust sound |
| **Unarmed strike** | Fist/punch impact |
| **Bow draw + release** | String tension → snap (5 bow variants: short, recurve, composite, military, ether) |
| **Arrow flight** | Whistling projectile travel |
| **Spell cast** | Magical charge-up during chambering, release on delivery |
| **Thrown object** | Whoosh with weight |
| **Consumable use** | Autoinjector click/hiss (HP green, MP blue) |
| **Skill book read** | Page turn / magical absorption |

## Combat - Impact & Outcomes

| Action | Notes |
|--------|-------|

| **Hit (magical)** | Elemental sizzle/crackle |

| **Miss** | Whiff / air swing |
| **Evade** | Quick dodge swoosh |
| **Parry** | Metal-on-metal clash |
